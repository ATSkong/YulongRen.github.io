研究方向

以下是我目前关注和探索的研究领域。随着学习的深入，内容将不断更新和完善。

FedTG: Feedback-driven Dynamic Task Graph for Tool Use
一个面向大规模 API 调用的自反思层次化智能体框架。通过将复杂用户查询的解决过程建模为一个动态可扩展的 DAG 任务图，每个子任务内部通过蒙特卡洛树搜索（MCTS）在工具空间中探索最优解，并允许子任务内部通过工具链组合搜索（Tool Composition Search, TCS）自动发现多步 API 调用路径。
